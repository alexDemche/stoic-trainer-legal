1. ЯКІ ДАНІ МИ ЗБИРАЄМО

Для коректної роботи додатка та персоналізації досвіду ми можемо збирати наступні дані:

  • Ім'я або Нікнейм: Щоб звертатися до вас у додатку.
  • Дата народження: Для аналітики вікових груп та відображення функції "Memento Mori" (візуалізація часу життя). Це поле є опціональним.
  • Ігровий прогрес: Рівень, звання, бали мудрості, статистика практик.
  • Технічні дані: ID пристрою для синхронізації (якщо ви використовуєте гостьовий режим).

2. ЯК МИ ВИКОРИСТОВУЄМО ДАНІ

Ваші дані використовуються виключно для:

  • Відображення вашого профілю та статистики.
  • Синхронізації прогресу між пристроями.
  • Покращення стабільності роботи додатка.

3. ЗБЕРІГАННЯ ТА БЕЗПЕКА

Ми використовуємо захищені сервери для зберігання вашого прогресу. Ми не продаємо та не передаємо ваші особисті дані третім особам.

4. ВИДАЛЕННЯ АКАУНТУ

Ви маєте право видалити свій акаунт та всі пов'язані з ним дані в будь-який момент через налаштування профілю.

Шлях: Профіль -> Налаштування -> Видалити акаунт.

Ця дія є незворотною.
